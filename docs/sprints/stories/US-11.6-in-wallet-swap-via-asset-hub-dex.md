## Goal

A user can swap Polkadot-ecosystem tokens on the Asset Hub DEX from inside the
wallet, with a live quote and single-confirm execution.

## Background

Asset Hub is Polkadot's system parachain for assets; its on-chain DEX
(`assetConversion`) lets users swap ecosystem tokens directly. It is integrated
as one **SwapService handler** (PRD [FR-8](../../PRD.md#functional-requirements), EPIC-2) — a new provider
is a new handler, never a new swap-UI branch — and quote/fee data is aggregated
via the Services SDK backend
([AD-24](../../ARCHITECTURE.md#architecture-decisions)). Execution is a Substrate
extrinsic on Asset Hub.

Materializes [FR-108](../../PRD.md#functional-requirements). **Retroactive** — already shipped.

## Acceptance criteria

- [ ] **AC-1** — **Given** a pair tradable on the Asset Hub DEX, **When** the user
  enters an amount, **Then** a quote (rate, fee, slippage, min received) is shown.
- [ ] **AC-2** — **Given** a quote, **When** the user confirms, **Then** the swap
  extrinsic is signed, submitted, and tracked in history.
- [ ] **AC-3** — **Given** insufficient liquidity or a stale quote, **When** the
  user swaps, **Then** a clear error is shown and nothing is submitted.
