## Goal

A user can perform a non-custodial cross-chain swap via SimpleSwap from inside
the wallet, reaching pairs across chains without surrendering custody, with a
live quote and single-confirm execution.

## Background

SimpleSwap is a **non-custodial, cross-chain** swap provider: it quotes and
settles exchanges across chains while the user retains custody. It is integrated
as one **SwapService handler** (PRD [FR-8](../../PRD.md#functional-requirements), EPIC-2) — a new provider
is a new handler, never a new swap-UI branch — and quote/fee data is aggregated
via the Services SDK backend
([AD-24](../../ARCHITECTURE.md#architecture-decisions)). SimpleSwap is a **keyed
provider**: its API is proxied through the SubWallet backend per
[NFR-16](../../PRD.md#non-functional-requirements), never shipped in the bundle. Because it settles across
chains, it participates in the cross-chain routing layer
([US-11.8](US-11.8-cross-chain-swap-routing.md)); the bridge step that routing
composes around it is delegated to ParaSpell
([AD-18](../../ARCHITECTURE.md#architecture-decisions)).

Materializes [FR-107](../../PRD.md#functional-requirements). **Retroactive** — already shipped.

## Acceptance criteria

- [ ] **AC-1** — **Given** a SimpleSwap-supported cross-chain pair, **When** the
  user enters an amount, **Then** a quote (rate, fee, slippage, min received,
  estimated time) is shown.
- [ ] **AC-2** — **Given** a quote, **When** the user confirms, **Then** the swap
  is signed, submitted to SimpleSwap, and tracked in history to settlement.
- [ ] **AC-3** — **Given** no route or a SimpleSwap outage / stale quote, **When**
  the user swaps, **Then** a clear error is shown and nothing is submitted.

## Dev notes

### Architecture constraints

- [AD-24](../../ARCHITECTURE.md#architecture-decisions) — quotes/fees via the Services SDK backend.
- [AD-18](../../ARCHITECTURE.md#architecture-decisions) — the bridge step in cross-chain routes is delegated to ParaSpell; this provider consumes routing, it does not reimplement XCM.
- Keyed provider: API proxied through the backend ([NFR-16](../../PRD.md#non-functional-requirements)), never bundled.
- Plugs into SwapService (FR-8); no SimpleSwap-specific swap-UI branch.

### Cross-story dependencies

- Builds on [US-2.4](US-2.4-swapservice-routing-engine.md) — the SwapService engine (FR-8) this handler registers into.
- Sibling: [US-11.8](US-11.8-cross-chain-swap-routing.md) — SimpleSwap's cross-chain pairs feed the multi-hop routing layer.

### Points justification

  provider) behind the shared handler, with keyed-proxy access and cross-chain
  settlement-tracking surface area.
