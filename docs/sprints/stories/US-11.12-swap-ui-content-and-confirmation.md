## Goal

The part of swap the user actually touches: the entry point, the form, the quote screen, the terms
they accept, the confirmation they read before signing, and the wording throughout. Provider-neutral
— every handler in this epic surfaces through it.

## Status

> **✅ done — all 17 rows below are settled**: 14 delivered, 3 closed without shipping. It carries
> **no FR**: FR-103 … FR-113 are per-provider and owned by

> not carry an unfinished row ([AGENTS.md](../../../AGENTS.md) rule 9).

## Scope

Folded in from the former one-issue-per-story maintenance ledger on 2026-07-23. It exists because
**EPIC-11's FR map is provider-shaped and this work is not**: no functional requirement says
"the swap screen", so seventeen issues about the swap screen had no capability to belong to. Filing
each of them under whichever provider happened to be selected at the time would have been a fiction.

**Out of scope:** the quote itself and the service that produces it —
[US-11.13](US-11.13-swap-quote-pipeline-and-service.md). This story owns what the user reads; that
one owns what the number means.
