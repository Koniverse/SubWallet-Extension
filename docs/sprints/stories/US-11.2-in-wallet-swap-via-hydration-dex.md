## Goal

A user can swap Polkadot-ecosystem tokens on the Hydration omnipool from inside
the wallet, with a live quote and single-confirm execution.

## Status

> **✅ done — shipped in 1.1.55.** All acceptance criteria are ticked and the 10 rows below are
> settled: nine delivered, one closed without shipping.

## Background

Hydration (formerly HydraDX) is a Substrate omnipool DEX, integrated as one
**SwapService handler** (PRD [FR-8](../../PRD.md#functional-requirements), EPIC-2). Quote/fee data is
aggregated via the Services SDK backend
([AD-24](../../ARCHITECTURE.md#architecture-decisions)); execution is a Substrate
extrinsic. Adding the provider is a new handler, not a swap-UI branch.

Materializes [FR-104](../../PRD.md#functional-requirements). **Retroactive** — already shipped.

## Acceptance criteria

- [x] **AC-1** — **Given** a pair tradable on Hydration, **When** the user enters an
  amount, **Then** a quote (rate, fee, slippage, min received) is shown.
- [x] **AC-2** — **Given** a quote, **When** the user confirms, **Then** the swap
  extrinsic is signed, submitted, and tracked in history.
- [x] **AC-3** — **Given** insufficient liquidity or a stale quote, **When** the user
  swaps, **Then** a clear error is shown and nothing is submitted.

## Tasks
